# Support

<p>
  <img src="../.github/assets/readme/section-bars/where-to-go.svg" alt="WHERE TO GO" width="100%">
</p>

| What you need | Where to go |
|---|---|
| install or build problem | open a GitHub issue or PR with `python --version`, exact install command, and full stderr |
| evidence dispute | open an issue or PR with exact counter-evidence file paths |
| feature or lane request | open an issue or PR and anchor it to the current blocker or roadmap file it would change |
| general question | open an issue or start with `FAQ.md` and `../README.md` |
| benchmark or falsification question | start with `../proofs/ENGINEERING_BLOCKERS.md` and include the specific artifact you are disputing |
| security vulnerability | `architects@zer0pa.ai`; do not open a public issue |
| code of conduct report | `architects@zer0pa.ai` |
| commercial licensing | `architects@zer0pa.ai` |
| legal or licensing interpretation | `architects@zer0pa.ai`; include the relevant `../LICENSE` section reference |
| release automation or PyPI publisher setup | follow `../proofs/runbooks/TRUSTED_PUBLISHING_OPERATOR_STEPS.md`; this remains operator-only |
| docs correction | open a PR with the exact file path and the evidence that justifies the change |

`../LICENSE` is the legal source of truth for license terms. This support guide
is a routing summary only and is not legal advice.

<p>
  <img src="../.github/assets/readme/section-bars/what-gets-a-fast-response.svg" alt="WHAT GETS A FAST RESPONSE" width="100%">
</p>

- reproducible reports with exact commands, exact file paths, and exact stderr
- evidence disputes with a concrete counter-artifact or counter-run
- docs corrections that cite the current blocker or release-surface file being fixed

<p>
  <img src="../.github/assets/readme/section-bars/response-expectations.svg" alt="RESPONSE EXPECTATIONS" width="100%">
</p>

- acknowledge target: 48 hours
- initial triage target: 7 days
- security issues follow `../SECURITY.md` and are prioritized
- owner-gated items may remain paused until the required outside action happens

<p>
  <img src="../.github/assets/readme/section-bars/what-does-not-get-a-response-here.svg" alt="WHAT DOES NOT GET A RESPONSE HERE" width="100%">
</p>

- requests to bless the repo as release-ready without new proof
- vague disputes that do not include an evidence path
- requests to reinterpret historical bundles as current authority
- requests for legal interpretation without a cited `../LICENSE` section
