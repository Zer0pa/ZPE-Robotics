# Code Of Conduct

This repo expects direct, respectful, evidence-based collaboration.

## Expected Behavior

- discuss code, proofs, and evidence without personal attacks
- report failures honestly
- keep claims proportional to evidence
- respect private staging boundaries

## Unacceptable Behavior

- harassment or abusive language
- hiding or rewriting negative results to create a pass narrative
- posting secrets or private repo details in the wrong channel

## Reporting

Report code-of-conduct concerns privately to `architects@zer0pa.ai`.
